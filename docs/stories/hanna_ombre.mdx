### HTML Explained
```html
<v:oval style="width:56;height:56">
    <v:fill src="/Hanna.jpeg" style="width:56;height:56" type="frame">
    </v:fill>
</v:oval>
```
#### [v:oval reference](https://docs.microsoft.com/en-us/windows/win32/vml/msdn-online-vml-oval-element) <br/>
`v:oval` will create an oval shape with width & height of `56px`. Also filled it with image src using `frame` type.
